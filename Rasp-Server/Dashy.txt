##Dashy
I also installed Dashy on my Raspberry Pi, in order to get an easy access to all my various self hosted services
https://github.com/lissy93/dashy

#Create a dedicated dashy directory
$ sudo mkdir /opt/dashy

#Install and enable Docker
$ sudo pacman -S docker
$ sudo systemctl enable --now docker containerd

#Download my dashy config file
$ sudo curl https://raw.githubusercontent.com/Antiz96/Linux-Customisation/master/Rasp-Server/conf.yml -o /opt/dashy/conf.yml

#Pull and run the container
$ sudo docker run -d -p 8080:80 -v /opt/dashy/conf.yml:/app/public/conf.yml --name dashy --restart=always lissy93/dashy:arm32v7
(I'm using the dashy:arm32v7 image since the Raspberry Pi 4 is an ARM7 board but you can use the dashy:arm64v7 image for ARM8, the dashy:latest image for x86_64 or the dashy:multi-arch which should work on every architecture)

##Update procedure
$ sudo docker pull lissy93/dashy:arm32v7
$ sudo docker stop dashy
$ sudo docker rm dashy
$ sudo docker run -d -p 8080:80 -v /opt/dashy/conf.yml:/app/public/conf.yml --name dashy --restart=always lissy93/dashy:arm32v7
