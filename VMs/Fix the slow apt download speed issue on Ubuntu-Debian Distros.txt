##On Ubuntu/Debian distros, I sometime face an issue where the apt package manager is pretty to download stuff during update, upgrade, etc...
##This is how I fix it

https://www.techrepublic.com/article/how-to-fix-the-slow-apt-get-update-issue-on-linux-machines/

#Add those three lines at the bottom of the file
$ sudo vi /etc/sysctl.conf
[...]
net.ipv6.conf.all.disable_ipv6 = 1
net.ipv6.conf.default.disable_ipv6 = 1
net.ipv6.conf.lo.disable_ipv6 = 1

#Restart network service to apply
$ sudo systemctl restart networking
