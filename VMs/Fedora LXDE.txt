##Fedora LXDE
Just a quick reminder on how I install a minimalistic Fedora environnement for various needs

#Install Fedora with LXDE (regular user as administrator)
Partitiong :
550M EFI
4G Swap
Left ext4 root

#Update the system and reboot
$ sudo dnf upgrade --refresh 
$ reboot

#Install dependencies and various stuff
$ sudo dnf install htop neofetch firefox bash-completion man mlocate curl tmux vim

#Disable firewalld cause I do need it (and I need as much performance as possible)
$ sudo systemctl disable firewalld

#Install virtualbox addons (if needed)
Insert the addon disk and execute the "run" script with sudo privileges

#Install qemu-agent for Proxmox (if needeed)
$ sudo dnf install qemu-guest-agent
$ sudo systemctl start qemu-guest-agent

#Reboot to apply changes
$ reboot

#Panel
Menu
Separator
Taskbar and Application Launcher (extended) --> Firefox, LXTerminal, PCManFM, AbiWord
Virtual Desktops Manager
Volume Control
Clock
Application Launcher

#Add the logout app to the panel
$ sudo vim ~/.config/lxpanel/LXDE/panels/panel #Add the "Button" and "id" part
[...]
Plugin {
  type=launchbar
  Config {
    Button {
      id=lxde-logout.desktop
    }
  }
}

#Startup app
vboxclient (if needeed)
VMware User Agent (if needed)
Spice vdagent
Liveinst Setup
User folder update

#Hide Desktop icons
Desktop Preferences --> Desktop Icon --> Uncheck all

#Install personal dotfiles
$ curl https://raw.githubusercontent.com/Antiz96/Linux-Customisation/master/Dotfiles/General/bashrc%20\(Fedora-RedHat%20Based%20Distro\) -o ~/.bashrc
$ vim ~/.bashrc #Get rid of the "Other" section
$ curl https://raw.githubusercontent.com/Antiz96/Linux-Customisation/master/Dotfiles/General/tmux.conf -o ~/.tmux.conf
$ vim ~/.tmux.conf #Get rid of comments at the top of the file

#Replace LXDM by LightDM
$ sudo systemctl disable lxdm
$ sudo dnf install lightdm lightdm-gtk-greeter
$ sudo systemctl enable lightdm
$ reboot
$ sudo dnf remove lxdm #You actually need to reboot before removing lxdm, otherwise it will break
