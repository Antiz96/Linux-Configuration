##Ubuntu Server
Just a quick reminder on how I install a minimalistic Ubuntu Server environnement for various needs

#Base Install
Minimal Install (nothing checked)
Partitionning :
550 MB EFI
4G Swap
Left ext4 / (0% reserved block)

#Install useful packages
$ sudo apt install man bash-completion openssh-server dnsutils traceroute rsync zip unzip diffutils firewalld mlocate htop curl vim openssl telnet (chronyd)

#Enable/start/configure various things
$ sudo systemctl enable --now ssh
$ sudo updatdb
$ sudo mandb
$ sudo systemctl enable --now firewalld
$ sudo firewall-cmd --add-service="ssh" --permanent
$ sudo firewall-cmd --reload

#Install qemu-guest-agent (if needed, for promox)
$ sudo apt install qemu-guest-agent
$ sudo systemctl enable --now qemu-guest-agent

#Downlaod my .bashrc
$ curl https://raw.githubusercontent.com/Antiz96/Linux-Customisation/master/Dotfiles/General/bashrc%20\(Debian-Ubuntu%20Based%20Distro%20SERVER%20-%20WSL\) -o ~/.bashrc
$ vim ~/.bashrc #Get rid of TMUX things and flatpak in the fullupgrade alias

#Static IP Adress
https://www.rosehosting.com/blog/how-to-configure-static-ip-address-on-ubuntu-20-04/
$ sudo vim /etc/netplan/00-installer-config.yaml
[...]
network:
  ethernets:
    ens18:
      addresses: [192.168.1.199/24]
      gateway4: 192.168.1.254
      nameservers:
        addresses: [192.168.1.1]
  version: 2

#Reboot
$ sudo reboot
