##Debian LXDE
Just a quick reminder on how I install a minimalistic Debian environnement for various needs

#Install Debian with LXDE (nothing else, no standard utilities)
Partitiong :
550M EFI
4G Swap
Left ext4 root (0 reserved space)

#Put user in the sudo group
$ usermod -aG sudo rcandau

#Install dependencies and various stuff
$ sudo apt install build-essential dkms linux-headers-$(uname -r) bash-completion mlocate curl tmux vim xfce4-notifyd htop neofetch

#Install virtualbox addons (if needed)
Insert the addon disk
$ sudo sh /media/cdrom0/VBoxLinuxAdditions.run

#Reboot to apply changes
$ reboot

#Panel
Menu
Separator
Taskbar and Application Launcher (extended) --> Firefox ESR, LXTerminal, PCManFM, Mousepad
Virtual Desktops Manager
Volume Control
Clock
Application Launcher

#Add the logout app to the panel
$ sudo vim ~/.config/lxpanel/LXDE/panels/panel #Add the "Button" and "id" part
[...]
Plugin {
  type=launchbar
  Config {
    Button {
      id=lxde-logout.desktop
    }
  }
}

#Startup app
Pulseaudio
Polkit
vboxclient (if on virtualbox)
User folders update
ScreenLocker
AT-SPI D-Bus Bus
Xfce Notification Daemon

#Hide Desktop icons
Desktop Preferences --> Desktop Icon --> Uncheck all

#Install personal dotfiles
$ curl https://raw.githubusercontent.com/Antiz96/Linux-Customisation/master/Dotfiles/General/bashrc%20\(Debian-Ubuntu%20Based%20Distro\) -o ~/.bashrc
$ vim ~/.bashrc #Get rid of the flatpak part in the fullupgrade alias + delete all the "Other" section
$ curl https://raw.githubusercontent.com/Antiz96/Linux-Customisation/master/Dotfiles/General/tmux.conf -o ~/.tmux.conf
$ vim ~/.tmux.conf #Get rid of comments at the top of the file

#Install Wine (if needed)
https://www.linuxcapable.com/how-to-install-wine-on-debian-11-bullseye/
$ sudo dpkg --add-architecture i386
$ sudo apt-get update && sudo apt install wine32 wine64 -y
