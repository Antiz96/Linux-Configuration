## Install YAY & Pamac
$ sudo pacman -S git
$ cd /tmp
$ git clone https://aur.archlinux.org/yay.git
$ cd yay
$ makepkg -si
$ yay -S pamac

## Installing bluetooth support
$ sudo pacman -S bluez pulseaudio-bluetooth
Install bluez-utils-compat (aur) #That package contains deprecated tools that I need to make my USB-Dongle work. If you don't need them, just install bluez-utils via pacman
$ sudo systemctl start bluetooth
$ sudo systemctl enable bluetooth

## Dependency for Arch Linux Updates Indicator
$ sudo pacman -S pacman-contrib

## Keyboard Shortcuts
Super = Open ArcMenu
Super + F = Switch size of windows (Maximize/Minimize)
Super + D = Close the window
Super + E = Home directory
Super + C = Start calculator
Super + M = Display the desktop
Super + A = Overview
Super + L = Lock the screen
Super + T = gnome-terminal
CTRL + ALT + DEL = gnome-system-monitor
